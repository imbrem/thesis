# Isotope

Lean 4 formalization of the metatheory for *Categorical Imperative Programming: Type theory, refinement, and semantics for SSA*.

## Building

Requires [elan](https://github.com/leanprover/elan) (the Lean toolchain manager).

```sh
lake build
```

## Documentation

API docs are generated by CI and published to GitHub Pages. To build locally:

```sh
DOCGEN_SRC=file lake build Isotope:docs
```

(The `DOCGEN_SRC=file` is only needed for local builds with SSH git remotes.)
